## Como capturar os prints:

1. Execute: `npm run web`
2. Abra o navegador: `http://localhost:8082`
3. Pressione `F12` para abrir DevTools
4. Pressione `Ctrl + Shift + M` para ativar o modo mobile
5. Use `⋯ > Capture screenshot` para tirar prints de cada tela
